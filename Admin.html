工具列說明

按 F1 可隨時開啟這份說明。這些按鈕主要處理資料同步、備份、匯入與批次整理。

重新載入
重新從 Firestore 讀取影片、筆記與設定。適合在另一個頁面剛修改資料後使用。
匯出 JSON
把目前影片與筆記資料複製成 JSON 備份。剪貼簿不可用時會放到匯入文字框中。
匯入 JSON
開啟 JSON 貼上區。按「匯入到 Firestore」後，會把 JSON 寫入 Firestore。
有日期設已完成
批次把已有日期但尚未完成的影片改成已完成，並把進度設為 100%。執行前會再次確認。
尚未匯入 / 匯入狀態
