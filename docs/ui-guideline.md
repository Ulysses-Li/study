# UI Guideline
# study.cwli.dev 介面規範

## 風格

- Bootstrap 5
- 簡潔乾淨
- 深色 Navbar
- 白色內容區
- 卡片式 Dashboard
- 手機可閱讀

---

## index.html

主要區塊：

1. Navbar
2. 月曆區
3. 搜尋與篩選
4. 當日影片
5. 當日筆記
6. 學習統計

---

## Admin.html

主要區塊：

1. Admin Navbar
2. Dashboard 統計卡
3. Videos 管理表格
4. Notes 管理表格
5. JSON 匯入
6. JSON 匯出

---

## login.html

主要區塊：

1. Login Card
2. Email 欄位
3. Password 欄位
4. Login Button
5. Forgot Password Button
6. Error Message

---

## UI 提示

必須使用：

- Loading spinner
- Skeleton
- Toast
- Alert
- Modal

---

## 月曆顏色

```text
灰色：無資料
藍色：已安排課程
黃色：進行中
綠色：完成
紅色：有筆記但未完成
```
